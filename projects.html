Research Projects

PBR Relighting & LoD for 3DGS Assets

Thesis Project: Developed algorithms and an interactive engine plugin to address core limitations of 3DGS assets lacking PBR properties, relighting capabilities and LoD strategies. Implemented inverse rendering algorithms to decouple assets into materials and lighting components.

3DGS PBR Unity Inverse Rendering

RL-based Adaptive Sketch Generation

Developed an RL agent (Clipasso-RL) to autonomously generate concise, human-style sketches by learning an adaptive stopping policy. Designed MDP framework with CLIP embeddings and hybrid action space.

Reinforcement Learning CLIP Computer Vision

Paintopia: Multi-Agent Drawing System

Developed a Multi-Agent System for children's creative drawing, driven by collaboration of interactive agent, VLM agent, and generative agents. Integrated ComfyUI workflows with Flux-Kontext and LoRA models.

Multi-Agent iOS ComfyUI Swift

Quantitative Options Trading

Provincial-Level Innovation Project. Developed ML-based strategy for quantitative trading of SSE 50ETF options using LSTM, XGBoost, and CNN-based time-series encoding.
